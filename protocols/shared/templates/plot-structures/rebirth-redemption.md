

| STAGE I: SETUP - THE ORDINARY WORLD: False Identity |  |
| ----- | :---- |
| [negative] The ordinary world |  |
| [negative] May have a false sense of balance or complete imbalance. |  |
| [negative] We get a snapshot of the main character’s problem, before the adventure begins. |  |
| [negative] The opening Image represents the central struggle & tone of the story. Often mirrors the Closing Image. |  |
| [negative] The protagonist is presented with some unlikeable, inhumane qualities to put off the audience |  |
| [negative] But the protagonist has limited awareness, somehow disconnected from reality as well as his community and is incomplete or unfulfilled |  |
| [negative] Expand on the opening image. Present the main character’s world as it is, and what is missing in their life. |  |
| [negative] If the protagonist’s life stays the same, he/she will figuratively die. |  |
| [negative] In addition, the main character’s flaw, his/her problem that needs fixing over the course of the story is revealed (In many rebirth stories, it is not the main character’s flaw, but another central character’s flaw that is presented for him to resolve over the course of the story – for the character to ‘arc’). |  |
| [negative] The protagonist is living in a false identity; he believes he is not worthy but instead pretends that he simply doesn’t care. |  |
| [neutral] The protagonist’s goal here is general; they want to pass a test, get to school, go to work, etc; something ‘normal’. |  |
| [neutral] Theme Stated (5) (during the Set-up) – The message, the truth you want to reveal by the end of your story, what your story is about in a larger sense. Usually, it is spoken to the main character or in their presence, but they don’t understand this truth…not until they go on the journey to find it. |  |
| [turning-point] Anticipation |  |
| [negative] Foreshadows: Look for essential details of your story and then plant some elements of anticipation like artifacts, people, or situations. |  |
| [negative] The protagonist’s life is oppressive and intolerable because he is alone and even though he may have a lot, he refuses to accept that his reality is miserable. |  |
| [negative] The hero seeks gratification that is often harmful to others, such as bullying; community suffers. |  |
| [turning-point] TURNING POINT \#1: Opportunity |  |
| [marker] INCITING INCIDENT - CALL TO ADVENTURE |  |
| [negative] he receives some sort of sign or warning urging him to change |  |
| [positive] He/she is given supernatural or visionary direction, or a secret message as to the goal; ONLY HE/SHE can solve it |  |
| [negative] The hero ignores this warning |  |
| [negative] The state of the world deteriorates; the protagonist now has increased awareness that something has to change |  |

| STAGE II: DREAM STAGE - GLIMPSE OF TRUE SELF - Initial Success |  |
| ----- | :---- |
| [negative] Opening balance deteriorates |  |
| [negative] tension begins to rise |  |
| [negative] B Story: discussion about the Theme – the nugget of truth. This discussion often takes place between the main character and an ally |  |
| [positive] The Mentor awaits. |  |
| [negative] protagonist shows a glimpse of his true self, something which indicates warmer humanity, his potential to change |  |
| [negative] the protagonist receives a warning; the threat becomes visible |  |
| [negative] but he/she refuses the call; their weakness is revealed - fear, immaturity, ignorance, resistance to change; he is not quite ready for complete fulfillment; Can I face this challenge? Do I have what it takes? Should I go at all? |  |
| [positive] For a while, all may seem to go relatively well. The threat may even seem to have receded. |  |
| [shocking] STUNNING SURPRISE \#1: Something shocking comes out of the blue and in an instant, changes everything; the protagonist’s life will never be the same again; the protagonist’s destiny changes; the protagonist faces a test of character that ironically makes them more worthy to meet the mentor. |  |
| [turning-point] TURNING POINT \#2: CHANGE OF PLANS |  |
| [marker] DREAM STAGE - THE SPECIAL WORLD - THE JOURNEY |  |
| [positive] the Herald guides the protagonist to meet the Mentor |  |
| [positive] the protagonist meets the Mentor and begins overcoming his/her fear of change* |  |
| [negative] the Mentor informs the protagonist of the consequences if they do not change |  |
| [negative] the Mentor shares a prophecy with the protagonist |  |
| [negative] the prophecy is incomplete |  |
| [positive] the protagonist is presented with a sword |  |
| [negative] the protagonist refuses again and returns his/her mundane everyday: he/she is stuck in the ordinary world |  |
| [neutral] the protagonist is focused on solving a very simple problem but is enough to put them on the path to adventure |  |
| [negative] the protagonist’s goal is a façade at this stage; they want revenge, money, escape, keeping a job, achieving fame, avoiding ridicule, etc but this is not their true purpose |  |
| [negative] we learn about a painful past experience the protagonist had - the pain makes them fearful of becoming something more than they are now |  |
| [negative] The protagonist see something that reminds him/her about the need to change - he/she is overpowered by the shock but this time he/she feels the need to do something about it. |  |
| [positive] The Herald gives the protagonist some special piece of information. |  |
| [negative] the mentor warns the protagonist about going against their advice |  |
| [positive] FIRST EPIPHANY. The protagonist has a moment of genuine insight — they begin to see this new world, challenge, or situation more clearly. Something clicks that shifts their understanding, even if they cannot yet see the full picture |  |
| [positive] The protagonist receives new clothes, symbolic of a new identity. |  |
| [positive] protagonist finally answers the call and crosses the threshold into the special world and begins committing to change - the ‘new/special world’ can be anything from a totally new universe to just a completely different place on earth |  |
| [positive] a small group of allies may welcome the protagonist into the special world |  |

| STAGE III: INITIAL FASCINATION: Protagonist Vacillates Between False Identity and True Self |  |
| ----- | :---- |
| [positive] The protagonist starts to become more open to the possibility of change |  |
| [positive] At first the protagonist is excited or fascinated because the new world is puzzling or unfamiliar, but not a place where they feel at home |  |
| [negative] The hero seems powerless. |  |
| [negative] He encounters inner demons and temptations |  |
| [positive] But then, the protagonist is given the elixir - something miraculous that will bring them back from certain doom later on. |  |
| [positive] The protagonist makes preparations to change. |  |
| [positive] The Mentor has faith that the protagonist will be able to change. |  |
| [negative] Meanwhile, time is running out to change. |  |
| [negative] An ‘ally’ betrays the protagonist. |  |
| [negative] For a while all seems to be going downhill; and protagonist seems to be having trouble letting go of his old self |  |
| [negative] protagonist comes across several obstacles, crises and conflicts; but he is offered more chances and reasons to change |  |
| [negative] But gradually the mood changes to frustration, difficulty, or oppression |  |
| [positive] SECOND EPIPHANY. A deeper realisation — the protagonist now understands something fundamental about themselves or their situation that they were blind to before. This insight shifts their approach from reactive to purposeful |  |
| [turning-point] TURNING POINT \# 3: POINT OF NO RETURN |  |
| [marker] FRUSTRATION |  |
| [positive] Journey into the ‘underworld’, a symbolic underworld or dark place, confronting their deepest fears or flaws |  |
| [positive] Someone appears who shows him the consequences of his actions |  |
| [negative] But things continue to go wrong |  |
| [positive] The protagonist sees more signs why he needs to change |  |
| [negative] greater obstacles and complications; the protagonist is being reminded of why is the way he is (possibly a repressed emotional wound from the past) |  |
| [negative] The Hero begins to feel terrible and disgusted with his false identity; he is in a state of living death |  |
| [negative] approach to the inmost cave; preparing to face dark truths and destroy internal falsehoods |  |

| STAGE IV: NIGHTMARE STAGE - COMPLICATIONS - Protagonist gradually accepts true self |  |
| ----- | :---- |
| [negative] Greater crises |  |
| [marker] HIGHER STAKES |  |
| [negative] He suddenly realises that things will never be the same for him again |  |
| [negative] The dark power begins to dominate |  |
| [negative] He likely feels compelled to run away from the truth |  |
| [negative] It poses a serious threat to the hero’s survival and there seems only one outcome |  |
| [negative] The final nightmare battle approaches |  |
| [negative] A nightmare battle with all the odds in the ‘dark force’s’ favor; the battle signifies the battle to defeat the old self |  |
| [shocking] STUNNING SURPRISE \#2: another shock comes out of the blue and once again completely destroys the Hero’s plan for victory and announces that sweeping improvisation will now be required for the Hero to have any chance of beating the Adversary |  |
| [turning-point] TURNING POINT \# 4: MAJOR SETBACK |  |
| [negative] The protagonist is overwhelmed with despair and momentarily retreats to his false identity |  |
| [negative] the allies disagree on what to do next |  |
| [negative] this is his/her lowest point |  |
| [negative] the villain appears to have cornered the protagonist |  |

| STAGE V: FINAL PUSH: FINAL ORDEAL - Retreat to identity then returns fully to true self; the thrilling escape from death |  |
| ----- | :---- |
| [positive] But just when all seems lost comes the “reversal” - the Mentor or some other ally brings the protagonist back to life |  |
| [negative] The ‘dark power’ demonstrates its power again. |  |
| [positive] “New Information”: This information (perhaps about the elixir) functions as a second Catalyst that gives the main character the choice again, to pack up and go home, or give it one more try to complete the journey |  |
| [positive] THIRD EPIPHANY. The protagonist becomes enlightened, possibly via a ‘redemption figure’; As the hero emerges from their crisis, they have renewed independent strength; here they fully understand the truth they could not see when they set out on the journey |  |
| [positive] The protagonist ‘seizes the sword’ and now fully accepts the consequences of his/her new identity as he/she faces a final test |  |
| [positive] The hero surpasses the Mentor. |  |
| [turning-point] TURNING POINT \# 5: CLIMAX |  |
| [positive] The protagonist rededicates himself/herself to the task and ‘comes back from the dead’ for one last attempt to change |  |
| [negative] The Mentor fights the dark forces and loses - only the protagonist can defeat the dark forces |  |
| [positive] The Mentor sacrifices himself to save the protagonist |  |
| [positive] The allies pull out their own bag of tricks. |  |
| [positive] In the nick of time, the protagonist changes |  |
| [positive] The protagonist is now willing to sacrifice him/herself for the sake of the others |  |
| [positive] In a final attempt to defeat the protagonist, the ‘dark force’ accuses the protagonist of being the same as them, but the protagonist is not fooled |  |
| [positive] The protagonist understands the ‘dark force’s’ weakness |  |
| [positive] The protagonist steps beyond false identity forever; his/her existence is transformed forever |  |
| [positive] The protagonist discovers his/her own true power |  |

| STAGE VI: THE GOAL AND THE AFTERMATH: Final Union, Transformation Complete and Fulfillment |  |
| ----- | :---- |
| [positive] The community is liberated; balance is restored |  |
| [positive] The protagonist returns to his/her community with the elixir; it appears as though he/she has mastered himself/herself |  |
| [neutral] The question is how much the protagonist has learnt from his/her experiences… |  |
| [neutral] Has he/she changed or was it all ‘just a dream’? |  |
| [positive] The hero emerges to ultimately find fulfillment in reconnecting with humanity |  |
| [positive] Final Image: opposite of Opening Image, proving, visually, that a change has occurred within the character. |  |
