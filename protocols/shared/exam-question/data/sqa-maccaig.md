## Best Comparison Pairs

1. **Visiting Hour + Memorial** — both deal with loss/death, different responses (emotional distance vs persistent memory)
2. **Brooklyn Cop + Hotel Room** — both explore violence beneath civilisation, one individual, one societal
3. **Aunt Julia + Sounds of the Day** — both about loss, one gradual realisation, one sudden
4. **Basking Shark + Assisi** — both involve an unexpected encounter forcing reflection on humanity
