## EDUQAS Format Note

EDUQAS Section A (Post-1914) gives NO extract — students write from memory. This makes it different from Edexcel (which provides extracts).

## Student Recommendation

**Examiner favourites:**
- Social class/responsibility — asked twice, the play's core message
- Individual character studies — EDUQAS rotates through the cast, one per year

**Strong prediction candidates (never asked on EDUQAS):**
- Mrs Birling — never asked, rich character for class and gender
- Gerald — never the focus, complex character (privilege, deception, compassion)
- Eva Smith — never asked on ANY board as the direct focus
