# Rubric — AQA GCSE Literature, Paper 1 Section A (Shakespeare)

**Loads on top of:** `rubric-base.md`. Universal Sophicly rules apply unless overridden here.

**Used by:** `inline-coaching-engine-1.md` for tasks `task='exam_crib'` (Macbeth / Romeo & Juliet / Much Ado / Merchant / Henry V) + `task='polishing'` (AQA Lit P1 Section A redraft).

**Paper structure:** Single 50-minute extract-based question. 30 marks (AO1+AO2+AO3) + 4 marks AO4 (SPaG) = 34 total.

---

## ASSESSMENT OBJECTIVES + WEIGHTING (PAPER 1 SECTION A)

| AO | Marks | Description |
|---|---:|---|
| AO1 | ~12 | Read, understand, respond + textual references. |
| AO2 | ~12 | Analyse language / form / structure for meaning + effect. |
| AO3 | ~6 | Context — relationships between text and contexts. |
| AO4 | 4 | SPaG (Spelling / Punctuation / Grammar / Vocabulary). |

**Critical:** AO4 IS SPaG for Shakespeare (and modern texts). Different from Edexcel IGCSE where AO4 = Context.

**Pointer phrase:** *AQA Shakespeare AO4 rewards SPaG accuracy alongside the 30-mark analytical core.*

---

## LEVEL DESCRIPTORS (UPGRADE LADDER)

- **Level 3 (clear):** clear understanding, clear method-effect link, some context, mostly accurate SPaG.
- **Level 4 (clear, sustained):** sustained clear response, effective method analysis, clear context links.
- **Level 5 (thoughtful, developed):** thoughtful, developed analysis, effective method exploration, effective context integration.
- **Level 6 (critical, exploratory):** critical, exploratory, conceptualised. Judicious / exploratory method analysis. Convincing context integration.

**Pointer phrases (use verbatim):**
- *Level 4 needs effective method exploration.*
- *Level 5 rewards thoughtful, developed analysis.*
- *Level 6 rewards critical, exploratory, conceptualised response with judicious context integration.*

## TTECEA SHAPE (PER `rubric-base.md`)

For Shakespeare, all 7 TTECEA elements compulsory:

1. T (concept-only).
2. T+e (technique + quote, anchored Act/Scene line ref).
3. E1 (audience reaction 1).
4. C (close analysis of words inside quote).
5. E2 (audience reaction 2 — different from E1).
6. A (Author's Purpose — compulsory for Lit).
7. C (Context — AO3 woven anchor).

**Pointer phrase:** *AQA Shakespeare body paragraphs use full TTECEA — all 7 elements, with A compulsory and C woven not bolted.*

## CONTEXT (AO3) — JACOBEAN ANCHORS FOR MACBETH

Strong anchors (use date + named source):

- *James I's Daemonologie* (1597) — supernatural authenticity argument.
- *Reginald Scot's Discoverie of Witchcraft* (1584) — sceptical counter-tradition; lets audience read Banquo's ghost as Macbeth's guilt externalised.
- *Basilikon Doron* (1599) — divine right of kings codified.
- *Gunpowder Plot* (1605) — context of audience's anti-regicide raw nerve.
- *Genesis 3:19* (*"for dust thou art..."*) — Christian fall narrative anchoring "Tomorrow" soliloquy.

**Pointer phrase:** *AO3 for Macbeth lands strongest with date + named source linked to specific BP claim.*

For other Shakespeare texts (R&J, Much Ado, Merchant, Henry V), see per-text crib sheets for canonical AO3 anchors.

## SHAKESPEARE-SPECIFIC ANTI-PATTERNS

(Stack on top of base anti-patterns.)

- **Hamartia = "fatal flaw".** WRONG. Hamartia = *fatal error in action* (per `feedback_hamartia_not_fatal_flaw`). Multiple defensible readings for Macbeth: witches' misreading / Duncan-murder / acting on ambition / accepting LM's manhood-pressure.
- **Witches don't control Macbeth.** They prophesy; Macbeth chooses (per memory).
- **Post-hamartia events = "redemption windows".** WRONG. Banquo's ghost / sleepwalking / Birnam Wood are *visible accounting of silenced conscience*, not refusal-windows. Window closed at 1.7.
- **Lady Macbeth "unsex me" = sexual.** WRONG (per `feedback_no_sexual_references_in_student_materials`). Reframe as gender-role rejection / agency-claim within Jacobean social order.
- **Death-in-any-case fate frame** for R&J — applies, not "tragic destiny inevitable".

## CONTROLLING-CONCEPT EXEMPLAR (MACBETH)

For students stuck on conclusion-S2 (controlling concept), the canonical Macbeth controlling concept:

> *Unchecked ambition divorced from conscience destroys the moral order, severs the agent from himself, and consumes him from within. The only escape is humility before a divinely ordered hierarchy — and Macbeth refuses that escape at every turn.*
