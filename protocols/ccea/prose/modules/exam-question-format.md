# BOARD FORMAT MODULE — CCEA GCSE English Literature

**Board:** CCEA GCSE English Literature
**Papers:** Unit 1 (Prose — closed book) | Unit 2 (Drama + Poetry — open book with extracts)
**Format:** Either/Or, "Show how far you agree" phrasing

---

## QUESTION FORMAT RULES — NON-NEGOTIABLE

### Unit 1 — The Study of Prose (Closed Book, NO Extract)

1. **Phrasing:** Every question uses *"Show how far you agree that [statement about character/theme]."* This is CCEA's distinctive "agree/disagree" format. NEVER use "Starting with this extract..." (AQA), "Explore how..." (Edexcel/OCR), or "Write about..." (EDUQAS).

2. **NO extract.** Unit 1 is closed book. Students write entirely from memory. Do not include any passage from the text.

3. **Either/Or pair:** ALWAYS generate TWO questions per text. Option (a) is typically **character-focused**. Option (b) is typically **theme-focused**. The student chooses one.

4. **Bold keyword:** Each question highlights one keyword in **bold** — this is the debatable claim the student must address. E.g. "Show how far you agree that Lennie is **responsible** for what happens to him."

5. **Mark allocation:** 40 marks per question (20 marks AO1, 20 marks AO2).

6. **Debate format:** The question is deliberately arguable — there should be evidence both FOR and AGAINST the statement. Strong answers explore both sides.

### Unit 2 — Drama and Poetry (Open Book, option (b) has extract)

1. **Option (a):** Character-focused, NO extract. Same "Show how far you agree" format as Unit 1.
2. **Option (b):** Theme-focused WITH an extract (~2-3 pages, with page references). Students use the extract as a starting point but must range across the whole play.
3. **Mark allocation:** 40 marks.

---

## COMPLETE QUESTION STRUCTURE — Unit 1 Prose (No Extract)

1. **Text title** — clearly stated
2. **Option (a)** — Character question with bold keyword: *"Show how far you agree that [character] is [bold keyword]."*
3. **Option (b)** — Theme question with bold keyword: *"Show how far you agree that [theme statement with bold keyword]."*
4. **Mark allocation** — [40 marks] for whichever option chosen

## COMPLETE QUESTION STRUCTURE — Unit 2 Drama (With Extract for option b)

1. **Text title** — clearly stated
2. **Option (a)** — Character question, no extract: *"Show how far you agree that [character] is [bold keyword]."*
3. **Option (b)** — Theme question WITH extract and page references: *"[Extract: pp. X-Y, from 'opening line...' to '...closing line'] Show how far you agree that [theme statement with bold keyword]."*
4. **Mark allocation** — [40 marks]

---

## KEY CCEA DIFFERENCES

1. **The bold keyword is critical** — it defines the debate. Choose a word that is genuinely arguable (e.g. "responsible", "powerless", "cruel", "selfish", "heroic")
2. **Option (a) always character, option (b) always theme** — this is consistent across all CCEA papers
3. **Unit 1 is closed book** — students cannot refer to the text. Questions must be answerable from memory alone.
4. **Unit 2 is open book** — students have the text. Option (b) includes a specific extract with page references.

## GENERATING CCEA QUESTIONS — GUIDELINES

When creating CCEA-style questions:
- The bold keyword should be **debatable** — not obviously true or false
- Option (a) should focus on a SINGLE character
- Option (b) should make a claim about a THEME that spans the whole text
- Both options should allow students to argue for AND against
- For Unit 2 option (b), include specific page references for the extract
- Past paper data shows CCEA loves: "powerless", "responsible", "cruel", "selfish", "heroic", "difficult", "important", "strong"
